## If Something Goes Wrong

| Error | Fix |
|-------|-----|
| Batch file won't run | Run as Administrator (right-click) |
| Port 9092 in use | Close other Kafka instances |
| App still can't connect | Wait 10 more seconds, restart app |
| Java not found | Install JDK: https://adoptium.net/ |
